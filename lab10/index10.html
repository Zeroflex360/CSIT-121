XML and JavaScript

XML (Extensible Markup Language)

XML is a kind of computer language which is used to store and transport data in a structured format which could further help in breaking down certain data.

Uses of XML:

  • To store data and describe information
  • Can be used as a sub html format allowing creation of tags with different names
  • Used to seperate information from presentation by showing the relationship between a node and another node
  • Can be used to organize the information we are working with
  • Moving data between different systems or software

Example of XML:

                <movie>
                    <title> Hitman's Wife's Bodygaurd </title>
                    <date released> June 16th, 2021 </date released>
                    <genre> Action/Comedy </genre>
                </movie>

JavaScript

JavaScript is a type of programming language allows users to introduce and use appealing features or tools on web pages with the intent of making it more appealing.

How It Relates to HTML and CSS:

JavaScript works with HTML and CSS by adding and transforming interactivity to web pages in general.

Common JavaScript elements, properties, methods, and functions:

  • Elements: variables, functions
  • Properties: innerHTML, style
  • Methods: getElementById(), addEventListener()
  • Functions: alert(), console.log()
